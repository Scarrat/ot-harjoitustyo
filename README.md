## Ohjelmistotekniikan harjoitustyö

### Luolastopeli
Ohjelmasta on tarkoitus tulla luolastopeli, jossa mennään luolassa tutkimassa huoneita johonkin loppuun asti/loputtomasti.


### Dokumentaatio
[Määrittelydokumentti](https://github.com/Scarrat/ot-harjoitustyo/blob/main/dokumentaatio/maarittelydokumentti.md)
[Tuntikirjanpito](https://github.com/Scarrat/ot-harjoitustyo/tree/main/dokumentaatio/tuntikirjanpito.md)


### Ohje
-Aja jar-tiedosto komennolla "java -jar dungeon.jar".
-Ohjaa hahmoa(mustaa ympyrää) nuolinäppäimillä huoneisiin.
-Elämäsi laskevat tai nousevat huoneesta riippuen.
-Yksi huone on reitti ulos.
-Jos kuolet aloitat alusta.
-Jos voitat niin sulje peli ja aja jar uudestaan.
-Kartta on sama kuolemien välissä.
